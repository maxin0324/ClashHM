# Release builds must keep ArkTS symbol names stable.
#
# ClashHM relies on framework and IPC string contracts:
# - module.json5 srcEntry entries for UIAbility and VpnExtensionAbility
# - Want parameter names passed between the app and VPN Extension
# - CommonEvent parameter names for config chunks and native-core commands
# - NAPI export names and JSON fields exchanged with the native core
#
# ArkGuard property/toplevel/filename/export obfuscation can rename those
# contracts and make release builds fail while debug builds still work. Keep
# obfuscation disabled unless every affected contract is covered by explicit
# keep rules and verified on a real device.
-disable-obfuscation
